01 About Me

Where hardware meets
intelligence

I'm a first-year B.Tech Computer Science student at Gautam Buddha University, Greater Noida, with a deep fascination for systems that exist at the crossroads of software and the physical world. My work spans from training machine learning models to programming microcontrollers — because I believe the most compelling technology is the kind you can both see and touch.

I'm drawn to questions like: how do robots perceive and navigate space? How can data science reveal patterns invisible to the human eye? How do we build interfaces that feel inevitable? These questions drive me to learn continuously, build experimentally, and iterate relentlessly.

Currently exploring the overlaps between AI, robotics, and IoT — building a foundation to eventually create systems that are truly autonomous, adaptive, and impactful.

04 Projects

Things I've Built

A snapshot of my current explorations — each project is a step toward something larger.

01
ML Python
🧠

Sentiment Analysis Engine

A text classification model using Scikit-learn and NLTK to predict sentiment polarity in product reviews. Preprocessed raw text data with tokenization, stopword removal, and TF-IDF vectorization — achieving over 85% accuracy on a benchmark dataset.

Python Scikit-learn NLTK Pandas
02
IoT Arduino
🌡️

Smart Environment Monitor

An Arduino-based embedded system that continuously reads temperature, humidity, and air quality via DHT22 and MQ-135 sensors. Data is displayed in real time on an OLED screen, with threshold-based alerts triggered via a buzzer and LED indicators.

Arduino C++ DHT22 OLED
03
Data Visualization
📊

COVID-19 Data Insights Dashboard

A Python-based exploratory data analysis project using Pandas and Matplotlib to visualize global pandemic trends. Includes time-series plots, correlation heatmaps, and geo-distribution charts from a public WHO dataset, surfacing key epidemiological patterns.

Python Pandas Matplotlib NumPy
04
Robotics Algorithm
🤖

Maze-Solving Robot Simulation

A conceptual robotics project implementing the A* pathfinding algorithm in Python, simulating an autonomous agent navigating a dynamically generated grid maze. Visualizes the search frontier, optimal path, and dead ends in real time using Matplotlib animation.

Python A* Algorithm Matplotlib
